# 코드맵 · 바이브 코딩 학습 사이트

코딩 입문자를 위한 시각 학습 사이트입니다. 웹 작동 원리부터 API·인증·설계·Git·AI/LLM까지
150여 장의 개념 인포그래픽과 쉬운 비유로 풀었습니다.

## 주요 기능
- **개념 학습** — 58개 개념 카드(그림 + 핵심 정리 + 비유 + 적용 예시 + 확대 보기)
- **추천 로드맵** — 입문 → 기초 → 심화 3단계 학습 경로
- **복습 센터** — 간격 반복(1·3·7·14·30일) 복습 일정 자동 관리
- **개념 연결 지도** — 선행·후속 관계를 잇는 지식 그래프
- **비교·분석 표** — 헷갈리는 개념 쌍 비교
- **코드 실습실** — HTML/CSS/JS 즉시 미리보기 + 도전 과제(예제 8종)
- **자가진단 퀴즈** — 즉시 채점 + 오답 노트
- **북마크 / 진도 저장 / 라이트·다크 테마 / 학습 기록 내보내기·가져오기**

진도·북마크·복습·오답·실습 코드는 브라우저 localStorage에 저장됩니다.

## 구조
```
index.html      화면 구조 + 스타일
data.js         개념·로드맵·실습·비교·퀴즈·선행관계 데이터
app.js          렌더링 및 상호작용 로직
images/         개념 인포그래픽 (약 150장)
```

## 실행
별도 빌드가 필요 없는 정적 사이트입니다. `index.html`을 브라우저로 열면 동작합니다.

## 배포 (Vercel)
정적 사이트이므로 설정 없이 배포됩니다.
1. 이 폴더를 GitHub 저장소에 올립니다.
2. [vercel.com](https://vercel.com) → **Add New… → Project** → 해당 저장소 **Import**.
3. Framework Preset은 **Other**, 빌드 설정은 비워 둔 채 **Deploy**.
